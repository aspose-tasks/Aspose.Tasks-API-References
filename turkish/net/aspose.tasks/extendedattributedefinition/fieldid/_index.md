## ExtendedAttributeDefinition.FieldId property

Alır veya ayarlar, özel bir alanın proje kimliğine karşılık gelir. Bir sabitin dize gösterimini kullanın[`ExtendedAttributeTask`](../../extendedattributetask/) belirtmek için sınıf`FieldId` özellik.

```csharp
public string FieldId { get; set; }
```

### Notlar

Ayarlamanın tercih edilen yolu`FieldId` özellik yaratmaktır[`ExtendedAttributeDefinition`](../) gibi özel fabrika yöntemlerinden birini kullanarak[`CreateTaskDefinition`](../createtaskdefinition/) veya[`CreateLookupTaskDefinition`](../createlookuptaskdefinition/) .

### Örnekler

```csharp
customFieldDefinition.FieldId = Aspose.Tasks.ExtendedAttributeTask.Number10.ToString("D");
```
