## GanttBarStyle.ParentStyle property

Göreve özgü özel stil için ana (veya ortak) stili alır veya ayarlar.

```csharp
public GanttBarStyle ParentStyle { get; set; }
```

### Notlar

Görev, farklı üst stiller içeren birden fazla özel stile sahip olabilir. Örneğin, görevin "Kritik" üst stile sahip özel stile ve "Normal" üst stile sahip başka bir stile sahip olduğunu düşünün. Basitçe söylemek gerekirse, görev kritikse, ilk stil uygulanır. Görev kritik olmaktan çıkarsa, ikinci stil uygulanır (bu mantık Microsoft Project Professional'dan devralınır).
