## GanttBarStyle.ParentStyle property

Özel göreve özgü stil için üst (veya ortak) stili alır veya ayarlar.

```csharp
public GanttBarStyle ParentStyle { get; set; }
```

### Notlar

Görev, farklı üst stillere sahip birden çok özel stile sahip olabilir. Örneğin, "Kritik" ebeveyn stiliyle özel stili olan bir görev ve "Normal" ebeveyn stili olan başka bir stili düşünün. Basitçe söylemek gerekirse, görev kritikse, ilk stil uygulanır. Görev kritik değilse, ikinci stil uygulanır (bu mantık Microsoft Project Professional'dan devralınır).
