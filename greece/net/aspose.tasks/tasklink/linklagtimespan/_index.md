## TaskLink.LinkLagTimeSpan property

Λαμβάνει ή ορίζει τη διάρκεια καθυστέρησης, ανάλογα με το LagFormat.

```csharp
public TimeSpan LinkLagTimeSpan { get; set; }
```

### Εξαιρέσεις

| εξαίρεση | κατάσταση |
| --- | --- |
| ArgumentException | Όταν προσπαθείτε να ορίσετε την τιμή για τα TaskLinks όπου LagFormat είναι TimeUnitType.Percent. |

### Παρατηρήσεις

Η καθυστέρηση σύνδεσης μπορεί να είναι μια ποσοστιαία τιμή (Το LagFormat είναι TimeUnitType.Percent). Σε αυτήν την περίπτωση η διάρκεια υπολογίζεται ως ποσοστό της διάρκειας του PredTask. Διαφορετικά, η μέθοδος επιστρέφει την τιμή TimeSpan που αντιπροσωπεύει την καθυστέρηση του TaskLink.
