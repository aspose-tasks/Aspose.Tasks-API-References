## FontResolveEventArgs.ResolvedFontName property

Λαμβάνει ή ορίζει το όνομα της λυμένης γραμματοσειράς. Μπορεί να ρυθμιστεί ώστε να ελέγχει τις γραμματοσειρές που χρησιμοποιούνται για την απόδοση μιας προβολής.

```csharp
public string ResolvedFontName { get; set; }
```

### Επιστρεφόμενη Αξία

Όνομα της γραμματοσειράς που ζητήσατε εάν βρεθεί η γραμματοσειρά ή όνομα εναλλακτικής γραμματοσειράς ή null εάν δεν μπορεί να βρεθεί η γραμματοσειρά.
