## ExtendedAttributeDefinition.FieldId property

Λαμβάνει ή θέτει αντιστοιχεί στο αναγνωριστικό έργου ενός προσαρμοσμένου πεδίου. Χρησιμοποιήστε αναπαράσταση συμβολοσειράς μιας σταθεράς από[`ExtendedAttributeTask`](../../extendedattributetask/) τάξη για να καθορίσετε`FieldId` ιδιοκτησία.

```csharp
public string FieldId { get; set; }
```

### Παρατηρήσεις

Προτιμότερος τρόπος ρύθμισης`FieldId` ιδιοκτησία είναι να δημιουργήσει[`ExtendedAttributeDefinition`](../) χρησιμοποιώντας μία από τις αποκλειστικές εργοστασιακές μεθόδους όπως[`CreateTaskDefinition`](../createtaskdefinition/) ή[`CreateLookupTaskDefinition`](../createlookuptaskdefinition/) .

### Παραδείγματα

customFieldDefinition.FieldId = Aspose.Tasks.ExtendedAttributeTask.Number10.ToString("D");
