## CreateLookupTaskDefinition(ExtendedAttributeTask, string) {#createlookuptaskdefinition_1}

Εργοστασιακή μέθοδος που δημιουργεί έναν εκτεταμένο ορισμό χαρακτηριστικών με αναζήτηση. Έχει[`CalculationType`](../calculationtype/) Ισούται μεLookup και μπορεί να χρησιμοποιηθεί μόνο στο Tasks. Πρέπει να καθορίσετε*fieldId* και*alias* όταν καλείτε αυτήν τη μέθοδο. Ο τύπος πεδίου προκύπτει από το πεδίο id.

```csharp
public static ExtendedAttributeDefinition CreateLookupTaskDefinition(ExtendedAttributeTask fieldId, 
    string alias)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| fieldId | ExtendedAttributeTask | Το καθορισμένο[`ExtendedAttributeTask`](../../extendedattributetask/) αναγνωριστικό πεδίου. |
| alias | String | Το καθορισμένοString ψευδώνυμο. |

### Επιστρεφόμενη Αξία

Δημιουργήθηκε το παράδειγμα του[`ExtendedAttributeDefinition`](../) τάξη με καθορισμένο*fieldId* και*alias*.

### Παραδείγματα

Χρησιμοποιήστε αυτό το παράδειγμα για να δημιουργήσετε έναν προσαρμοσμένο ορισμό πεδίου για μια εργασία με αναζήτηση και, στη συνέχεια, συμπληρώστε την με τιμές κειμένου:

```csharp
var taskTextAttr = ExtendedAttributeDefinition.CreateLookupTaskDefinition(ExtendedAttributeTask.Text27, "My custom field");
taskTextAttr.AddLookupValue(new Value { Id = 1, Val = "Text value 1", Description = "Text value description 1" });
taskTextAttr.AddLookupValue(new Value { Id = 2, Val = "Text value 2", Description = "Text value description 2" });
project.ExtendedAttributes.Add(taskTextAttr);
```
