## GanttBarStyle.ParentStyle property

Λαμβάνει ή ορίζει γονικό (ή κοινό) στυλ για προσαρμοσμένο στυλ για συγκεκριμένη εργασία.

```csharp
public GanttBarStyle ParentStyle { get; set; }
```

### Παρατηρήσεις

Το Task μπορεί να έχει πολλά προσαρμοσμένα στυλ με διαφορετικά γονικά στυλ. Για παράδειγμα, σκεφτείτε ότι η εργασία έχει προσαρμοσμένο στυλ με "Κρίσιμο" γονικό στυλ και ένα άλλο στυλ με "Κανονικό" γονικό στυλ. Με απλά λόγια, εάν η εργασία είναι κρίσιμη, εφαρμόζεται το πρώτο στυλ. Εάν η εργασία γίνει μη κρίσιμη, εφαρμόζεται το δεύτερο στυλ (αυτή η λογική έχει κληρονομηθεί από το Microsoft Project Professional).
