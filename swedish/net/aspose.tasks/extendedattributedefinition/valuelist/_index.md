### Anmärkningar

När värden för utökade attribut anges som egenskaper för element i schemat, kan de antingen specificeras med värden eller genom referenser till värdena i denna lista. Applikationer kan anta ordningsföljd av listan genom ordning som anges här. För närvarande stöds för MSP 2003/2007 Xml och MSP 2003 mpp-format. Ändra inte denna lista direkt. Använd metoderna ExtendedAttributeDefinition.AddLookupValue/RemoveLookupValue istället.
