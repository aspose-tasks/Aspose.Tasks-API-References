## ExtendedAttributeDefinition.FieldId property

Hämtar eller sätter motsvarar projekt-id:t för ett anpassat fält. Använd strängrepresentation av en konstant från[`ExtendedAttributeTask`](../../extendedattributetask/) klass att specificera`FieldId` egenskap.

```csharp
public string FieldId { get; set; }
```

### Anmärkningar

Föredraget sätt att ställa in`FieldId` egendom är att skapa[`ExtendedAttributeDefinition`](../) med en av de dedikerade fabriksmetoderna som[`CreateTaskDefinition`](../createtaskdefinition/) eller[`CreateLookupTaskDefinition`](../createlookuptaskdefinition/) .

### Exempel

```csharp
customFieldDefinition.FieldId = Aspose.Tasks.ExtendedAttributeTask.Number10.ToString("D");
```
