## ProjectServerSaveOptions.Timeout property

Hämtar eller ställer in timeout som används när man väntar på bearbetning av spara projektbegäran av en Project Servers köbehandlingstjänst. Standardvärdet för den här egenskapen är 1 minut.

```csharp
public TimeSpan Timeout { get; set; }
```

### Anmärkningar

Behandlingstiden kan vara längre för stora projekt eller om Project Server-instansen är för upptagen med att svara på andra förfrågningar.
