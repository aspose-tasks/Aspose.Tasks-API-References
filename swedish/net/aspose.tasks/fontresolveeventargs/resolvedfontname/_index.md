## FontResolveEventArgs.ResolvedFontName property

Hämtar eller ställer in namnet på löst teckensnitt. Kan ställas in för att styra teckensnitt som används för att rendera en vy.

```csharp
public string ResolvedFontName { get; set; }
```

### Returvärde

Namn på det begärda teckensnittet om teckensnittet hittats eller namnet på reservtypsnittet eller null om teckensnittet inte kan hittas.
