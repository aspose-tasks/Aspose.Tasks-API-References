## Project.CriticalPath property

Hämtar en samling som innehåller en lista över kritiska uppgifter som utgör Kritisk väg för detta projekt. Detta är en O(n) operation, där n är antalet uppgifter i projektet.

```csharp
public TaskCollection CriticalPath { get; }
```

### Returvärde

en samling som representerar en lista över alla kritiska uppgifter.
