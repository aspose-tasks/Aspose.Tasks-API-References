Innehåller klasser för att arbeta med databaser.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [DbSettings](./dbsettings/) | Gör det möjligt att ange inställningar som ska läsas från projektdatabasen. |
| [MpdSettings](./mpdsettings/) | Gör det möjligt att ställa in nödvändiga alternativ för att läsa projektdata från MPD-format (MS Access-databasfilformat). |
| [MspDbSettings](./mspdbsettings/) | Gör det möjligt att ställa in nödvändiga alternativ för att läsa projektdata från MS Project Server-databasen. |
| [PrimaveraDbSettings](./primaveradbsettings/) | Gör det möjligt att ställa in nödvändiga alternativ för att läsa projektdata från Primavera-databasen. |
