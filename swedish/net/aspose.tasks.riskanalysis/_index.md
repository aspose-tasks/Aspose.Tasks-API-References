Innehåller klasser för att skapa riskanalysrapport.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [RiskAnalysisResult](./riskanalysisresult/) | Representerar ett resultat av riskanalys. |
| [RiskAnalysisSettings](./riskanalysissettings/) | Anger inställningar för att utföra riskanalys. |
| [RiskAnalyzer](./riskanalyzer/) | Utför en Monte Carlo-simulering baserat på de specificerade riskanalysinställningarna. |
| [RiskItemStatistics](./riskitemstatistics/) | Representerar ett objekt som lagrar statistisk data för uppgiften för det analyserade projektet. |
| [RiskItemStatisticsCollection](./riskitemstatisticscollection/) | Representerar en samling som innehåller instanserna av[`RiskItemStatistics`](../aspose.tasks.riskanalysis/riskitemstatistics/) class. |
| [RiskPattern](./riskpattern/) | Representerar ett riskmönster för en projektuppgift. |
| [RiskPatternCollection](./riskpatterncollection/) | Representerar en samling som innehåller instanserna av[`RiskPattern`](../aspose.tasks.riskanalysis/riskpattern/) class. |
## Uppräkning

| Uppräkning | Beskrivning |
| --- | --- |
| [ConfidenceLevel](./confidencelevel/) | Anger stödda konfidensnivåer som används i riskanalys som motsvarar procentandelen av tiden de faktiska värdena kommer att ligga inom optimistiska och pessimistiska uppskattningar. |
| [ProbabilityDistributionType](./probabilitydistributiontype/) | Anger typer av sannolikhetsfördelningar som stöds. |
| [RiskItemType](./riskitemtype/) | Anger uppgiftsfält för vilka den statistiska informationen samlas in under riskanalys. |
