## GanttBarStyle.ParentStyle property

Hämtar eller ställer in överordnad (eller gemensam) stil för anpassad uppgiftsspecifik stil.

```csharp
public GanttBarStyle ParentStyle { get; set; }
```

### Anmärkningar

Task kan ha flera anpassade stilar med olika överordnade stilar. Överväg till exempel att uppgiften har anpassad stil med "Kritisk" överordnad stil och en annan stil med "Normal" överordnad stil. Enkelt uttryckt, om uppgiften är kritisk, tillämpas den första stilen. Om uppgiften blir icke-kritisk tillämpas den andra stilen (denna logik ärvs från Microsoft Project Professional).
