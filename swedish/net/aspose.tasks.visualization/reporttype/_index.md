## ReportType enumeration

Typ av projektets grafiska rapport.

```csharp
public enum ReportType
```

### Värderingar

| namn | Värde | Beskrivning |
| --- | --- | --- |
| ProjectOverview | `0` | Visar projektets start- och slutdatum, procent av varaktigheten som är klar, procentandel färdig för uppgifter på toppnivå och kommande milstolpar. |
| CostOverview | `1` | Visar projektets start- och slutdatum, aktuell schemalagd och återstående kostnad, % komplett och kostnadsvärden för uppgifter på toppnivå. |
| WorkOverview | `2` | Visar baslinje, faktisk, återstående arbete för varje toppnivåuppgift och arbete för arbetsresurser. |
| ResourceOverview | `3` | Visar baslinje, faktisk och återstående arbete per resurs. |
| ResourceCostOverview | `4` | Visar baslinje, faktisk och återstående kostnad per resurs. |
| CriticalTasks | `5` | Visar projektuppgifter som är kritiska. |
| LateTasks | `6` | Visar projektuppgifter som är försenade. |
| Milestones | `7` | Visar milstolpar som är sena, kommer upp och slutförs. |
| UpcomingTask | `8` | Visar uppgifter som ska ske under innevarande vecka och uppgifter som börjar under innevarande vecka. |
| CostOverruns | `9` | Visar kostnadsvariationer efter uppgift och resurs. |
| TaskCostOverview | `10` | Visar baslinje, faktisk och återstående kostnad för alla uppgifter på toppnivå. |
| OverallocatedResources | `11` | Visar antalet återstående arbetstimmar för övertilldelade resurser. |
| SlippingTasks | `12` | Visar uppgifter som ska slutföras efter deras slutdatum för baslinjen (baslinjen måste anges). |
| BestPracticeAnalyzer | `13` | Visar uppgifter utan faktiskt arbete, inte tilldelade uppgifter, uppgifter med varaktighet mindre än 8 timmar och sammanfattningar tilldelade med resurserna. |
| Burndown | `14` | Inkluderar diagram för nedbränning av arbete och uppgiftsnedbrytning. Diagrammet för arbetsnedbrytning visar hur mycket arbete människor har slutfört, hur mycket som är planerat att vara klart före projektets slutdatum, och baslinjeuppskattningen av hur mycket arbete som skulle slutföras vid denna tidpunkt i projektet. Uppgiftsnedbrytningsdiagrammet visar antalet slutförda uppgifter, antalet återstående, och baslinjeuppskattningen av hur många som skulle vara klara vid denna tidpunkt i projektet. |
| CashFlow | `15` | Visar kostnaderna och de ackumulerade kostnaderna per kvartal för alla uppgifter på toppnivå. |
