## Aspose.Tasks för .NET

{{% alert color="primary" %}}

![Aspose.Tasks för .NET-produktlogotypen](../home_1.png)

Aspose.Tasks för .NET är ett klassbibliotek som gör det möjligt för .NET-applikationer att läsa och skriva MPP- eller P6XML-filer utan att använda Microsoft Project eller Oracle Primavera P6. Det är för närvarande den enda .NET-komponenten som tillhandahåller denna funktionalitet. Aspose.Tasks för .NET är en mogen produkt som erbjuder stabilitet och flexibilitet. Som med alla Aspose-filhanteringskomponenter fungerar Aspose.Tasks för .NET bra med både Win Form- och WebForm-applikationer.

Med Aspose.Tasks för .NET är det möjligt att programmera med Microsoft Project (MPP/XML) filer utan Microsoft Office Automation. Komponenten ger ett lättanvänt API för att manipulera projektfiler, vilket sparar tid och pengar jämfört med att utveckla liknande funktioner från grunden.

## Aspose.Tasks för Java

{{% alert color="primary" %}}

![Aspose.Tasks för Java-produktlogotypen](../home_2.png)

Aspose.Tasks för Java är ett klassbibliotek som gör det möjligt för Java-applikationsutvecklare att bädda in funktionaliteten för att bearbeta MPP/MPT/XLS-dokument utan att ens använda Microsoft Project. API:et eliminerar därför kravet på Microsoft Office Automation i en sådan applikation. Aspose.Tasks för Java är hämtat från motsvarande .NET-produkt och är en mogen produkt som erbjuder stabilitet och flexibilitet. Komponenten ger ett lättanvänt API för att manipulera projektfiler, vilket sparar tid och pengar jämfört med att utveckla liknande funktioner från grunden.
