## GanttBarStyle.ParentStyle property

Ottiene o imposta lo stile principale (o comune) per lo stile specifico dell'attività personalizzata.

```csharp
public GanttBarStyle ParentStyle { get; set; }
```

### Osservazioni

L'attività può avere più stili personalizzati con diversi stili principali. Ad esempio, considera l'attività con uno stile personalizzato con lo stile principale "Critico" e un altro stile con lo stile principale "Normale". In poche parole, se l'attività è critica, viene applicato il primo stile. Se l'attività diventa non critica, viene applicato il secondo stile (questa logica è ereditata da Microsoft Project Professional).
