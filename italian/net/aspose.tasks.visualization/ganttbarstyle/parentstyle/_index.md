## GanttBarStyle.ParentStyle property

Ottiene o imposta lo stile padre (o comune) per uno stile specifico dell'attività personalizzato.

```csharp
public GanttBarStyle ParentStyle { get; set; }
```

### Osservazioni

L'attività può avere più stili personalizzati con diversi stili padre. Ad esempio, considera un'attività con uno stile personalizzato con stile padre "Critico" e un altro stile con stile padre "Normale". In poche parole, se l'attività è critica, viene applicato il primo stile. Se l'attività diventa non critica, viene applicato il secondo stile (questa logica viene ereditata da Microsoft Project Professional).
