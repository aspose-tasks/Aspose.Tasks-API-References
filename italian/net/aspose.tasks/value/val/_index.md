## Value.Val property

Ottiene o imposta il valore effettivo nella rappresentazione interna. Preferisci utilizzare le proprietà fortemente tipizzate elencate di seguito.

```csharp
public string Val { get; set; }
```

### Osservazioni

Se vuoi impostare il valore del testo, preferisci usare fortemente tipizzato[`StringValue`](../stringvalue/) proprietà.

Se si desidera impostare il valore del numero o del costo, è preferibile utilizzare una tipizzazione forte[`NumericValue`](../numericvalue/) proprietà.

Se si desidera impostare i valori Data/Inizio/Fine, è preferibile utilizzare caratteri fortemente tipizzati[`DateTimeValue`](../datetimevalue/) proprietà.

Se si desidera impostare il valore Durata, è preferibile utilizzare una tipizzazione forte[`Duration`](../duration/) proprietà.

Se il tuo tipo non era elencato, usa`Val` proprietà.
