## ResourceAssignment.MakeTPs method

Genera un elenco di dati in scala cronologica.

```csharp
public DateTime MakeTPs(DateTime start, TimeSpan time, Calendar calendar, 
    List<TimephasedData> list, bool isWorking, int type)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| start | DateTime | La data di inizio specificata. |
| time | TimeSpan | L'orario di lavoro specificato. |
| calendar | Calendar | Il calendario lavorativo specificato. |
| list | List`1 | L'elenco dei dati in scala cronologica. |
| isWorking | Boolean | Il flag specificato che specifica se i dati rapportati alla scala cronologica funzionano o meno. |
| type | Int32 | Il tipo di dati rapportato alla scala cronologica specificato. |

### Valore di ritorno

Una data massima dall'elenco o una data di inizio se l'elenco è vuoto.
