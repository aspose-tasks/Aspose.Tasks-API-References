## SaveOptions.RenderToSinglePage property

Ottiene o imposta un valore che indica se il rendering di un progetto deve essere eseguito su una singola pagina quando il progetto viene salvato in formato grafico. La dimensione della pagina verrà modificata in modo che il progetto sottoposto a rendering possa essere contenuto in una pagina.

```csharp
public bool RenderToSinglePage { get; set; }
```
