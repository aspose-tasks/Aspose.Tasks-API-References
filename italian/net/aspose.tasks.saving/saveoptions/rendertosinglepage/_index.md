## SaveOptions.RenderToSinglePage property

Ottiene o imposta un valore che indica se è necessario eseguire il rendering di un progetto su una singola pagina quando il progetto viene salvato in formato grafico. Le dimensioni della pagina verranno modificate in modo che il progetto renderizzato possa adattarsi a una pagina.

public bool RenderToSinglePage { get; set; }
