## SaveOptions.View property

Ottiene o imposta un elenco delle colonne della vista di cui eseguire il rendering ([`GanttChartColumn`](../../../aspose.tasks.visualization/ganttchartcolumn/) ). Se non impostato, vengono visualizzati solo ID attività, nomi attività, inizio e fine. Se sia Visualizza che[`ViewSettings`](../viewsettings/)le proprietà sono impostate, le colonne di View sovrascrivono le colonne di ViewSettings.

public ProjectView View { get; set; }
