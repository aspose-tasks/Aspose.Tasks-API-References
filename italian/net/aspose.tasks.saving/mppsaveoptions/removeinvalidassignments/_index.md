## MPPSaveOptions.RemoveInvalidAssignments property

Ottiene o imposta un valore che indica se rimuovere le assegnazioni di risorse non valide durante il salvataggio in MPP. MS Project crea un'assegnazione di risorse vuota per ogni attività. Imposta questo flag su true per rimuoverli durante il salvataggio.

public bool RemoveInvalidAssignments { get; set; }
