## ReportType enumeration

项目图形报告的类型。

```csharp
public enum ReportType
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| ProjectOverview | `0` | 显示项目的开始和完成日期、 完成持续时间的百分比、顶级任务和即将到来的里程碑的完成百分比。 |
| CostOverview | `1` | 显示项目的开始日期和完成日期、当前 计划成本和剩余成本、顶级任务的完成百分比和成本值。 |
| WorkOverview | `2` | 显示每个顶级任务的基线、实际、剩余工时和工时资源的工时。 |
| ResourceOverview | `3` | 按资源显示基线、实际和剩余工时。 |
| ResourceCostOverview | `4` | 按资源显示基准、实际和剩余成本。 |
| CriticalTasks | `5` | 显示关键的项目任务。 |
| LateTasks | `6` | 显示延迟的项目任务。 |
| Milestones | `7` | 显示迟到、即将到来和已完成的里程碑。 |
| UpcomingTask | `8` | 显示本周到期的任务和本周开始的任务。 |
| CostOverruns | `9` | 按任务和资源显示成本差异。 |
| TaskCostOverview | `10` | 显示所有顶级任务的基线、实际和剩余成本。 |
| OverallocatedResources | `11` | 显示超额分配资源的剩余工时数。 |
| SlippingTasks | `12` | 显示应在基准完成日期（必须设置基准）之后完成的任务。 |
| BestPracticeAnalyzer | `13` | 显示没有实际工时的任务、未分配的任务、持续时间少于 8 小时的 任务以及分配有资源的摘要。 |
| Burndown | `14` | 包括工作燃尽图和任务燃尽图。 工作燃尽图显示人们已经完成了多少工作，计划在项目完成日期之前完成多少， 以及此时将完成多少工作的基线估计在项目中。 任务燃尽图显示已完成的任务数、剩余任务数、 以及项目中此时将完成多少任务的基线估计。 |
| CashFlow | `15` | 显示所有顶级任务的每季度成本和累计成本。 |
