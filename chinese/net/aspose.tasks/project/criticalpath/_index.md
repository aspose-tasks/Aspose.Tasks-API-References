## Project.CriticalPath property

获取一个集合，其中包含构成该项目关键路径的关键任务列表。 这是一个 O(n) 操作，其中 n 是项目中的任务数。

```csharp
public TaskCollection CriticalPath { get; }
```

### 返回值

代表所有关键任务列表的集合。
