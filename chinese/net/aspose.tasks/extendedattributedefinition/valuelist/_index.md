## ExtendedAttributeDefinition.ValueList property

获取 List&lt;Value&gt; ValueList.

```csharp
public List<Value> ValueList { get; }
```

### 评论

当扩展属性的值被指定为架构中元素的属性时， 它们可以通过值或通过引用此列表中包含的值来指定。 应用程序可以通过此处指定的顺序假定列表的顺序。 当前支持 MSP 2003/2007 Xml 和 MSP 2003 mpp 格式。 不要直接更改此列表。请改用 ExtendedAttributeDefinition.AddLookupValue/RemoveLookupValue 方法。
