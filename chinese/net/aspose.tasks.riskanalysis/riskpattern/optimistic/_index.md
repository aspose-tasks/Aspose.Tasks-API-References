## RiskPattern.Optimistic property

获取或设置最可能发生的任务持续时间的百分比，它可能发生在最佳项目场景中。 默认值为 75，这意味着如果估计的指定任务持续时间为 4 天，则乐观持续时间将为 3 天。

public int Optimistic { get; set; }
