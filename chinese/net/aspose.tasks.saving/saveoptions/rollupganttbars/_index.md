## SaveOptions.RollUpGanttBars property

获取或设置一个值，该值指示是否应标记摘要任务栏上的子任务。 对于子任务，Rollup 字段指示是否将子任务甘特条上的信息汇总到摘要任务栏。 对于汇总任务，汇总字段指示汇总任务栏是否显示汇总条。 您必须将汇总任务的汇总字段设置为是，以便任何子任务汇总到它们。

```csharp
public bool RollUpGanttBars { get; set; }
```
