## SaveOptions.FitTimescaleToEndOfPage property

获取或设置视图的日历部分是否应呈现到最后一页的末尾（右侧）。 如果值为 false，日历部分将呈现到 EndDate，即使页面上有空白区域。

```csharp
public bool FitTimescaleToEndOfPage { get; set; }
```
