## CreateResourceDefinition(CustomFieldType, ExtendedAttributeResource, string) {#createresourcedefinition}

Microsoft Project では「なし」と表示される単純な拡張属性定義を作成するファクトリ メソッド。 [`CalculationType`](../calculationtype/)に等しいNoneリソースのみで使用できます。 指定する必要があります*customFieldType*、*fieldId*と*alias*このメソッドを呼び出すと.

```csharp
public static ExtendedAttributeDefinition CreateResourceDefinition(CustomFieldType customFieldType, 
    ExtendedAttributeResource fieldId, string alias)
```

| --- | --- | --- |
| customFieldType | CustomFieldType | 指定された[`CustomFieldType`](../../customfieldtype/)タイプ。 |
| fieldId | ExtendedAttributeResource | 指定された[`ExtendedAttributeResource`](../../extendedattributeresource/)フィールド ID。 |
| alias | String | 指定されたStringエイリアス。 |

### 戻り値

の作成されたインスタンス[`ExtendedAttributeDefinition`](../)指定されたクラス*customFieldType*、*fieldId*と*alias*.

### 例

次の例を使用して、カスタム テキスト フィールド定義を作成します:

```csharp

project.ExtendedAttributes.Add(resourceTextAttr);
```
