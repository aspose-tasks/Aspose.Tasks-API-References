## TaskLink.LinkLagTimeSpan property

LagFormat. に応じてラグ期間を取得または設定します

```csharp
public TimeSpan LinkLagTimeSpan { get; set; }
```

### 例外

| 例外 | 調子 |
| --- | --- |
| ArgumentException | LagFormat が TimeUnitType.Percent である TaskLinks の値を設定しようとしたとき。 |

### 備考

リンク ラグはパーセンテージ値にすることができます (LagFormat は TimeUnitType.Percent). この場合、期間は PredTask の期間のパーセンテージとして計算されます. それ以外の場合、メソッドは TaskLink のラグを表す TimeSpan 値を返します。
