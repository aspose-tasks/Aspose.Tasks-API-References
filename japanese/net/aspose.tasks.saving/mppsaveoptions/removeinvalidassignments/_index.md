## MPPSaveOptions.RemoveInvalidAssignments property

MPP への保存時に無効なリソース割り当てを削除するかどうかを示す値を取得または設定します。 MS Project は、タスクごとに空のリソース割り当てを作成します。保存時にそれらを削除するには、このフラグを true に設定します。

public bool RemoveInvalidAssignments { get; set; }
