## SaveOptions.TasksFilter property

ガント チャート、タスク シート、およびタスク配分状況チャートに表示されるタスクをフィルター処理するために使用される条件を取得または設定します。

```csharp
public ICondition<Task> TasksFilter { get; set; }
```

### 備考

値が指定されていない場合、非表示のタスク (つまり、折りたたまれたタスクの子孫タスク) を削除するデフォルトのフィルターが使用されます。
