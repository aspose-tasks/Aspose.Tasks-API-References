## .NET 用の Aspose.Tasks

{{% alert color="primary" %}}

![Aspose.Tasks for .NET 製品のロゴ](../home_1.png)

Aspose.Tasks for .NET は、.NET アプリケーションが Microsoft Project または Oracle Primavera P6 を使用せずに MPP または P6XML ファイルを読み書きできるようにするクラス ライブラリです。 現在、この機能を提供する唯一の .NET コンポーネントです。 Aspose.Tasks for .NET は、安定性と柔軟性を提供する成熟した製品です。 すべての Aspose ファイル管理コンポーネントと同様に、Aspose.Tasks for .NET は Win Form アプリケーションと WebForm アプリケーションの両方でうまく機能します。

Aspose.Tasks for .NET を使用すると、Microsoft Office Automation を使用せずに Microsoft Project (MPP/XML) ファイルでプログラミングできます。 このコンポーネントは、プロジェクト ファイルを操作するための使いやすい API を提供し、同様の機能をゼロから開発する場合に比べて時間とお金を節約します。
