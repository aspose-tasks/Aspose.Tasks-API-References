## CreateTaskDefinition(CustomFieldType, ExtendedAttributeTask, string) {#createtaskdefinition}

Fabrieksmethode die een eenvoudige uitgebreide attribuutdefinitie creëert, die Microsoft Project weergeeft als "Geen".
