## GanttBarStyle.ParentStyle property

Krijgt of stelt bovenliggende (of gemeenschappelijke) stijl in voor aangepaste taakspecifieke stijl.

```csharp
public GanttBarStyle ParentStyle { get; set; }
```

### Opmerkingen

Taak kan meerdere aangepaste stijlen hebben met verschillende bovenliggende stijlen. Stel bijvoorbeeld dat een taak een aangepaste stijl heeft met de bovenliggende stijl "Kritiek" en een andere stijl met de bovenliggende stijl "Normaal". Simpel gezegd, als de taak kritiek is, wordt de eerste stijl toegepast. Als de taak niet-kritiek wordt, wordt de tweede stijl toegepast (deze logica is overgenomen van Microsoft Project Professional).
