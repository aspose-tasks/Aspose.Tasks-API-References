## Aspose.Tasks voor .NET

{{% alert color="primary" %}}

![Aspose.Tasks voor .NET-productlogo](../home_1.png)

Aspose.Tasks voor .NET is een klassenbibliotheek waarmee .NET-toepassingen MPP- of P6XML-bestanden kunnen lezen en schrijven zonder Microsoft Project of Oracle Primavera P6 te gebruiken. Het is momenteel de enige .NET-component die deze functionaliteit biedt. Aspose.Tasks voor .NET is een volwassen product dat stabiliteit en flexibiliteit biedt. Zoals met alle bestandsbeheercomponenten van Aspose, werkt Aspose.Tasks voor .NET goed met zowel Win Form- als WebForm-applicaties.

Met Aspose.Tasks voor .NET is het mogelijk om te programmeren met Microsoft Project (MPP/XML) bestanden zonder Microsoft Office Automation. De component biedt een eenvoudig te gebruiken API voor het manipuleren van projectbestanden, waardoor tijd en geld worden bespaard in vergelijking met het helemaal opnieuw ontwikkelen van vergelijkbare functies.

{{% /alert %}}

## Aspose.Tasks voor Java

{{% alert color="primary" %}}

![Aspose.Tasks voor Java-productlogo](../home_2.png)

Aspose.Tasks voor Java is een klassenbibliotheek waarmee Java-toepassingsontwikkelaars de functionaliteit van het verwerken van MPP/MPT/XLS-documenten kunnen integreren zonder Microsoft Project te gebruiken. De API elimineert dus de vereiste van Microsoft Office Automation in een dergelijke toepassing. Aspose.Tasks voor Java is overgezet van het equivalente .NET-product en is een volwassen product dat stabiliteit en flexibiliteit biedt. De component biedt een eenvoudig te gebruiken API voor het manipuleren van projectbestanden, waardoor tijd en geld worden bespaard in vergelijking met het helemaal opnieuw ontwikkelen van vergelijkbare functies.
