## PrimaveraCriticalActivitiesDefiningMethod enumeration

Specifies the method for defining critical activities in Primavera.

## Members
| Member name | Description |
| :- | :- |
|TOTAL_FLOAT|Activities with TotalSlack (Total Float in Primavera) less than or equal to the value of [critical_total_float_limit](/tasks/python-net/aspose.tasks/primaveraprojectproperties/) are marked as critical.|
|LONGEST_PATH|Activities on the longest path are marked as critical.|
