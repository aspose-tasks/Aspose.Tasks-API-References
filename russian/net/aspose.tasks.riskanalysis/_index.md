Содержит классы для создания отчета об анализе рисков.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [RiskAnalysisResult](./riskanalysisresult/) | Представляет результат анализа риска. |
| [RiskAnalysisSettings](./riskanalysissettings/) | Указывает параметры для выполнения анализа рисков. |
| [RiskAnalyzer](./riskanalyzer/) | Выполняет моделирование методом Монте-Карло на основе указанных настроек анализа рисков. |
| [RiskItemStatistics](./riskitemstatistics/) | Представляет элемент, в котором хранятся статистические данные для задачи анализируемого проекта. |
| [RiskItemStatisticsCollection](./riskitemstatisticscollection/) | Представляет коллекцию, содержащую экземпляры[`RiskItemStatistics`](../aspose.tasks.riskanalysis/riskitemstatistics/) класс. |
| [RiskPattern](./riskpattern/) | Представляет шаблон риска для задачи проекта. |
| [RiskPatternCollection](./riskpatterncollection/) | Представляет коллекцию, содержащую экземпляры[`RiskPattern`](../aspose.tasks.riskanalysis/riskpattern/) класс. |
## перечисление

| перечисление | Описание |
| --- | --- |
| [ConfidenceLevel](./confidencelevel/) | Определяет поддерживаемые уровни достоверности, используемые в анализе рисков, которые соответствуют проценту времени, в течение которого фактические значения будут находиться в пределах оптимистичных и пессимистичных оценок. |
| [ProbabilityDistributionType](./probabilitydistributiontype/) | Указывает типы поддерживаемых вероятностных распределений. |
| [RiskItemType](./riskitemtype/) | Указывает поля задачи, для которых собирается статистическая информация во время анализа рисков. |
