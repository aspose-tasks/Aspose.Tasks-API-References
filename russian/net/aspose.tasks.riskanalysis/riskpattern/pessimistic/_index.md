## RiskPattern.Pessimistic property

Получает или задает процент наиболее вероятной продолжительности задачи, которая может произойти в наихудшем из возможных сценариев проекта. Значение по умолчанию — 125, что означает, что если расчетная указанная продолжительность задачи составляет 4 дня, то пессимистичная продолжительность будет равна 5 дням.

```csharp
public int Pessimistic { get; set; }
```
