## RiskPattern.Optimistic property

Получает или задает процент наиболее вероятной продолжительности задачи, которая может произойти в наилучшем возможном сценарии проекта. Значение по умолчанию — 75, что означает, что если расчетная указанная продолжительность задачи составляет 4 дня, оптимистичная продолжительность будет равна 3 дням.

```csharp
public int Optimistic { get; set; }
```
