## MPPSaveOptions.RemoveInvalidAssignments property

Получает или задает значение, указывающее, следует ли удалять недопустимые назначения ресурсов при сохранении в MPP. MS Project создает пустое назначение ресурсов для каждой задачи. Установите для этого флага значение true, чтобы удалить их при сохранении.

```csharp
public bool RemoveInvalidAssignments { get; set; }
```
