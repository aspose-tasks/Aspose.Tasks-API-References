## MPPSaveOptions.ProtectionPassword property

Получает или устанавливает пароль, который используется для защиты результирующего файла MPP. В настоящее время поддерживается для MS Project 2010 и более новых форматов. Значение Null указывает, что файл проекта не защищен.

```csharp
public string ProtectionPassword { get; set; }
```
