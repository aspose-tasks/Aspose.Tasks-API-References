## SvgOptions.UseGradientBrush property

Определяет, использовать ли градиентную кисть при рендеринге макета проекта. В настоящее время использование градиентной кисти не поддерживается для рендеринга в SVG.

```csharp
public override bool UseGradientBrush { get; set; }
```
