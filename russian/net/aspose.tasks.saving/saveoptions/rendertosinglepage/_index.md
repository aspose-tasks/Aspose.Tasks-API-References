## SaveOptions.RenderToSinglePage property

Получает или задает значение, указывающее, должен ли проект отображаться на одной странице при сохранении проекта в графическом формате. Размер страницы будет изменен, чтобы отображаемый проект мог уместиться на одной странице.

```csharp
public bool RenderToSinglePage { get; set; }
```
