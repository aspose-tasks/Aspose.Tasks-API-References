## SaveOptions.TasksFilter property

Получает или задает условие, которое используется для фильтрации задач, отображаемых на диаграммах Ганта, листе задач и диаграммах использования задач.

public ICondition<Task> TasksFilter { get; set; }

### Примечания

Если значение не указано, используется фильтр по умолчанию, который удаляет невидимые задачи -- т.е. задачи-потомки свернутых задач.
