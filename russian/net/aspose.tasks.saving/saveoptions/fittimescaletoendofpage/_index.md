## SaveOptions.FitTimescaleToEndOfPage property

Получает или задает, должен ли раздел календаря представления отображаться до конца (справа) последней страницы. Если значение равно false, раздел календаря отображается точно до EndDate, даже если на странице есть пустое место.

```csharp
public bool FitTimescaleToEndOfPage { get; set; }
```
