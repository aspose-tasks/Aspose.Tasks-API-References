## CalculationType enumeration

Определяет тип расчета значения пользовательского атрибута.

```csharp
public enum CalculationType
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Означает, что расширенный атрибут не имеет таблицы поиска формулы и просто хранит значение, установленное пользователем. |
| Lookup | `1` | Означает, что значение расширенного атрибута ограничено значениями из таблицы поиска. |
| Formula | `2` | Означает, что значение расширенного атрибута рассчитывается по формуле, определенной в[`Formula`](../extendedattributedefinition/formula/) . |
