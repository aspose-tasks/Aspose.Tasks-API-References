## CalendarException.EnteredByOccurrences property

Получает или задает значение, указывающее, определяется ли диапазон повторения путем ввода количества вхождений. False указывает, что диапазон повторения определяется путем ввода даты окончания.

public bool EnteredByOccurrences { get; set; }
