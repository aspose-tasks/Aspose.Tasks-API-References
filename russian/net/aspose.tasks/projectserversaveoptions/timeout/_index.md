## ProjectServerSaveOptions.Timeout property

Получает или задает время ожидания, используемое при ожидании обработки запроса на сохранение проекта службой обработки очереди Project Server. Значение по умолчанию для этого свойства — 1 минута.

```csharp
public TimeSpan Timeout { get; set; }
```

### Примечания

Время обработки может быть больше для больших проектов или в случае, когда экземпляр Project Server слишком занят, отвечая на другие запросы.
