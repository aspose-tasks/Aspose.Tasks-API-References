## ProjectServerSaveOptions.ProjectName property

Получает или задает имя проекта, которое отображается в списке проектов Project Server\Project Online. Должно быть уникальным в экземпляре Project Server\Project Online. Если значение опущено, вместо него будет использоваться значение свойства Prj.Name.

```csharp
public string ProjectName { get; set; }
```
