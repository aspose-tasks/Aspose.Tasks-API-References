## FontResolveEventArgs.ResolvedFontName property

Получает или задает имя разрешенного шрифта. Может быть установлен для управления шрифтами, используемыми для визуализации вида.

```csharp
public string ResolvedFontName { get; set; }
```

### Возвращаемое значение

Имя запрошенного шрифта, если шрифт найден или имя резервного шрифта или null, если шрифт не найден.
