## Calendar.Make24HourCalendar method

```csharp
public static Calendar Make24HourCalendar(Calendar calendar)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| calendar | Calendar | Календарь, из которого можно сделать 24-часовой календарь. |
