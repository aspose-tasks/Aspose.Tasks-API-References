## CalculationMode enumeration

Определяет режим расчета проекта.

```csharp
public enum CalculationMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Нет. Даты и стоимость проекта в этом режиме не пересчитываются. |
| Automatic | `1` | Автоматический режим. Даты и стоимость проекта пересчитываются при использовании этого режима. |
| Manual | `2` | Ручной режим. В этом режиме пересчитываются только необходимые поля, например UID и ID объектов. |
