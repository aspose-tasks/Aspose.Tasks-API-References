## ExtendedAttributeDefinition.RemoveLookupValue method

Удаляет значение из внутреннего списка поиска. Это предпочтительный способ для манипуляций с[`ValueList`](../valuelist/) .

```csharp
public void RemoveLookupValue(Value value)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | Value | Значение для удаления из поиска. |

### Примечания

Этот метод работает только для[`ExtendedAttributeDefinition`](../) instances , которые имеют[`CalculationType`](../calculationtype/) равноLookup .
