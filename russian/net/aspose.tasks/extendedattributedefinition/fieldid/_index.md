## ExtendedAttributeDefinition.FieldId property

Получает или устанавливает соответствие идентификатору проекта пользовательского поля. Использовать строковое представление константы из[`ExtendedAttributeTask`](../../extendedattributetask/) класс для указания`FieldId` свойство.

```csharp
public string FieldId { get; set; }
```

### Примечания

Предпочтительный способ установки`FieldId` свойство состоит в том, чтобы создавать[`ExtendedAttributeDefinition`](../) с помощью одного из специальных фабричных методов, таких как[`CreateTaskDefinition`](../createtaskdefinition/) или[`CreateLookupTaskDefinition`](../createlookuptaskdefinition/) .

### Примеры

```csharp
customFieldDefinition.FieldId = Aspose.Tasks.ExtendedAttributeTask.Number10.ToString("D");
```
