## Aspose.Tareas para .NET

{{% alert color="primary" %}}

![Aspose.Tasks para el logotipo del producto .NET](../home_1.png)

Aspose.Tasks for .NET es una biblioteca de clases que permite que las aplicaciones .NET lean y escriban archivos MPP o P6XML sin usar Microsoft Project u Oracle Primavera P6. Actualmente es el único componente .NET que proporciona esta funcionalidad. Aspose.Tasks para .NET es un producto maduro que ofrece estabilidad y flexibilidad. Al igual que con todos los componentes de administración de archivos de Aspose, Aspose.Tasks para .NET funciona bien con las aplicaciones Win Form y WebForm.

Con Aspose.Tasks for .NET, es posible programar con archivos de Microsoft Project (MPP/XML) sin Microsoft Office Automation. El componente proporciona una API fácil de usar para manipular archivos de proyectos, ahorrando tiempo y dinero en comparación con el desarrollo de funciones similares desde cero.

## Aspose.Tareas para Java

{{% alert color="primary" %}}

![Logotipo del producto Aspose.Tasks para Java](../home_2.png)

Aspose.Tasks for Java es una biblioteca de clases que permite a los desarrolladores de aplicaciones Java integrar la funcionalidad de procesamiento de documentos MPP/MPT/XLS sin siquiera usar Microsoft Project. La API, por lo tanto, elimina el requisito de Microsoft Office Automation en dicha aplicación. Portado desde su producto .NET equivalente, Aspose.Tasks para Java es un producto maduro que ofrece estabilidad y flexibilidad. El componente proporciona una API fácil de usar para manipular archivos de proyectos, ahorrando tiempo y dinero en comparación con el desarrollo de funciones similares desde cero.
