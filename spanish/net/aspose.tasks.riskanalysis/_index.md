Contiene clases para crear el Informe de análisis de riesgos.

## Clases

| Clase | Descripción |
| --- | --- |
| [RiskAnalysisResult](./riskanalysisresult/) | Representa un resultado del análisis de riesgo. |
| [RiskAnalysisSettings](./riskanalysissettings/) | Especifica la configuración para realizar el análisis de riesgos. |
| [RiskAnalyzer](./riskanalyzer/) | Realiza una simulación de Monte Carlo basada en la configuración de análisis de riesgo especificada. |
| [RiskItemStatistics](./riskitemstatistics/) | Representa un elemento que almacena datos estadísticos para la tarea del proyecto analizado. |
| [RiskItemStatisticsCollection](./riskitemstatisticscollection/) | Representa una colección que contiene las instancias del[`RiskItemStatistics`](../aspose.tasks.riskanalysis/riskitemstatistics/) clase. |
| [RiskPattern](./riskpattern/) | Representa un patrón de riesgo para una tarea de proyecto. |
| [RiskPatternCollection](./riskpatterncollection/) | Representa una colección que contiene las instancias del[`RiskPattern`](../aspose.tasks.riskanalysis/riskpattern/) clase. |
## Enumeración

| Enumeración | Descripción |
| --- | --- |
| [ConfidenceLevel](./confidencelevel/) | Especifica los niveles de confianza admitidos utilizados en el análisis de riesgos que corresponden a el porcentaje de tiempo en que los valores reales estarán dentro de las estimaciones optimistas y pesimistas. |
| [ProbabilityDistributionType](./probabilitydistributiontype/) | Especifica los tipos de distribuciones de probabilidad compatibles. |
| [RiskItemType](./riskitemtype/) | Especifica campos de tareas para los que se recopila información estadística durante el análisis de riesgos. |
