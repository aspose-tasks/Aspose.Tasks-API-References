## MPPSaveOptions.RemoveInvalidAssignments property

Obtiene o establece un valor que indica si se eliminan las asignaciones de recursos no válidas al guardar en MPP. MS Project crea una asignación de recursos vacía para cada tarea. Establezca este indicador en verdadero para eliminarlos al guardar.

public bool RemoveInvalidAssignments { get; set; }
