## MPPSaveOptions.ProtectionPassword property

Obtiene o establece una contraseña que se utiliza para proteger un archivo MPP resultante. Actualmente es compatible con MS Project 2010 y formatos más nuevos. El valor nulo indica que el archivo del proyecto no está protegido.
