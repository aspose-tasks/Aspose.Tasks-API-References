## SaveOptions.RenderToSinglePage property

Obtiene o establece un valor que indica si un proyecto debe representarse en una sola página cuando el proyecto se guarda en formato gráfico. El tamaño de la página cambiará para que el proyecto representado pueda caber en una sola página.

public bool RenderToSinglePage { get; set; }
