## SaveOptions.FitTimescaleToEndOfPage property

Obtiene o establece si una sección de calendario de una vista debe representarse al final (lado derecho) de la última página. Si el valor es falso, la sección de calendario se representa exactamente en EndDate, incluso si hay un espacio vacío en una página.

```csharp
public bool FitTimescaleToEndOfPage { get; set; }
```
