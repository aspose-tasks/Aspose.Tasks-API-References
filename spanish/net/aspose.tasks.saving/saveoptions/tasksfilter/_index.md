## SaveOptions.TasksFilter property

Obtiene o establece la condición que se usa para filtrar tareas representadas en diagramas de Gantt, hoja de tareas y uso de tareas.

```csharp
public ICondition<Task> TasksFilter { get; set; }
```

### Observaciones

Si no se especifica el valor, se usa el filtro predeterminado que elimina las tareas no visibles, es decir, las tareas descendientes de las tareas colapsadas.
