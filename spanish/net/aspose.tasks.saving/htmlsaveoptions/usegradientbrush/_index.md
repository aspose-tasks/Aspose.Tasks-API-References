## HtmlSaveOptions.UseGradientBrush property

Obtiene o establece un valor que indica si se debe usar el pincel de degradado al renderizar el diseño del proyecto. Actualmente, no se admite el uso de un pincel degradado cuando se renderiza a HTML.

```csharp
public override bool UseGradientBrush { get; set; }
```
