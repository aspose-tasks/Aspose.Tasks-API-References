Contiene clases para trabajar con Bases de Datos.

## Clases

| Clase | Descripción |
| --- | --- |
| [DbSettings](./dbsettings/) | Permite especificar la configuración para leer desde la base de datos del proyecto. |
| [MpdSettings](./mpdsettings/) | Permite configurar las opciones necesarias para leer los datos del proyecto en formato MPD (formato de archivo de base de datos de MS Access). |
| [MspDbSettings](./mspdbsettings/) | Permite configurar las opciones necesarias para leer los datos del proyecto desde la base de datos de MS Project Server. |
| [PrimaveraDbSettings](./primaveradbsettings/) | Permite configurar las opciones necesarias para leer los datos del proyecto desde la base de datos Primavera. |
