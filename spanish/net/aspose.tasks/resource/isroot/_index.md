## Resource.IsRoot property

Obtiene el indicador que indica si el recurso es un recurso raíz. El recurso raíz es un recurso especial que está destinado a admitir los formatos internos de MS Project y no está destinado a ser utilizado directamente desde el código del usuario.

```csharp
public virtual bool IsRoot { get; }
```
