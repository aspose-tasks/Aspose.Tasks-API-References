## TableField.WrapText property

Obtiene o establece un valor que indica si el texto de la columna se puede ajustar a varias líneas o si se debe truncar cuando excede el ancho de la columna. Compatible con la versión MSP 2010 y posteriores.

public bool WrapText { get; set; }
