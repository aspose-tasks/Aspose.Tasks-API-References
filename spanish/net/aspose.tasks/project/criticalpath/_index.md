## Project.CriticalPath property

Obtiene una colección que contiene una lista de tareas críticas que comprenden la ruta crítica de este proyecto. Esta es una operación O(n), donde n es el número de tareas en el proyecto.

```csharp
public TaskCollection CriticalPath { get; }
```

### Valor_devuelto

una colección que representa una lista de todas las tareas críticas.
