## FontResolveEventArgs.ResolvedFontName property

Obtiene o establece el nombre de la fuente resuelta. Se puede configurar para controlar las fuentes utilizadas para representar una vista.

```csharp
public string ResolvedFontName { get; set; }
```

### Valor_devuelto

Nombre de la fuente solicitada si se encuentra la fuente o nombre de la fuente alternativa o nulo si no se encuentra la fuente.
