## FilterCriteria.CriteriaRows property

Obtiene la lista de niños[`FilterCriteria`](../) filas. Si el filtro contiene más de una fila de criterios, el efecto de un operador Y es que se deben cumplir los criterios de ambas filas para que se muestre la tarea o el recurso como resultado de este filtro. El efecto de un operador O operador es que se deben cumplir los criterios para una u otra fila.
