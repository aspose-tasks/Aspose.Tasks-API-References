## ParseErrorCallback delegate

Representa una devolución de llamada de método para manejar los errores de análisis que pueden ocurrir al leer datos xml.

```csharp
public delegate object ParseErrorCallback(object sender, ParseErrorArgs args);
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| sender | Object | el objeto de origen del error de análisis. |
| args | ParseErrorArgs | la instancia de la[`ParseErrorArgs`](../parseerrorargs/) clase que contiene los datos del evento. |

### Valor_devuelto

el valor forzado para establecer en el objeto de remitente especificado.
