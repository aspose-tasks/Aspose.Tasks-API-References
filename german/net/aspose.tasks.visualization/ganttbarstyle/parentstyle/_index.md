## GanttBarStyle.ParentStyle property

Ruft übergeordneten (oder gemeinsamen) Stil für benutzerdefinierten aufgabenspezifischen Stil ab oder legt ihn fest.

```csharp
public GanttBarStyle ParentStyle { get; set; }
```

### Bemerkungen

Aufgabe kann mehrere benutzerdefinierte Stile mit unterschiedlichen übergeordneten Stilen haben. Stellen Sie sich beispielsweise vor, dass eine Aufgabe einen benutzerdefinierten Stil mit dem übergeordneten Stil „Kritisch“ und einen anderen Stil mit dem übergeordneten Stil „Normal“ hat. Einfach ausgedrückt, wenn die Aufgabe kritisch ist, wird der erste Stil angewendet. Wenn die Aufgabe unkritisch wird, wird der zweite Stil angewendet (diese Logik wird von Microsoft Project Professional geerbt).
