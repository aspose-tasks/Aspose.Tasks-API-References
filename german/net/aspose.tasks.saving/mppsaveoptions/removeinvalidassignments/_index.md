## MPPSaveOptions.RemoveInvalidAssignments property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob ungültige Ressourcenzuweisungen beim Speichern in MPP entfernt werden sollen. MS Project erstellt für jede Aufgabe eine leere Ressourcenzuweisung. Setzen Sie dieses Flag auf true, um sie beim Speichern zu entfernen.

public bool RemoveInvalidAssignments { get; set; }
