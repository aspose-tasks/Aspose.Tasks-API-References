## SaveOptions.TasksFilter property

Ruft die Bedingung ab oder legt sie fest, die verwendet wird, um Aufgaben zu filtern, die in Gantt-, Aufgabenblatt- und Aufgabenverwendungsdiagrammen gerendert werden.

```csharp
public ICondition<Task> TasksFilter { get; set; }
```

### Bemerkungen

Wenn kein Wert angegeben ist, wird der Standardfilter verwendet, der nicht sichtbare Aufgaben entfernt – dh untergeordnete Aufgaben von reduzierten Aufgaben.
