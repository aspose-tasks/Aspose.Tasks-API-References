## PdfSaveOptions.PageSavingCallback property

Ruft einen benutzerdefinierten Callback ab oder legt ihn fest, der verwendet wird, um einen Ausgabestrom für jede gerenderte Seite zu erhalten. Ist anwendbar, wenn[`SaveToSeparateFiles`](../savetoseparatefiles/) Option wird verwendet.

```csharp
public IPageSavingCallback PageSavingCallback { get; set; }
```
