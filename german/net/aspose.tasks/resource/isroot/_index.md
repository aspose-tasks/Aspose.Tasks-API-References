## Resource.IsRoot property

Ruft das Flag ab, das angibt, ob die Ressource eine Root-Ressource ist. Root-Ressource ist eine spezielle Ressource, die Interna von MS Project-Formaten unterstützen soll und nicht direkt aus dem Code des Benutzers verwendet werden soll.
