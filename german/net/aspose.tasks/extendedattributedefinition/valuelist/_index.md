## ExtendedAttributeDefinition.ValueList property

Ruft die Liste&lt;Wert&gt; ValueList ab.

```csharp
public List<Value> ValueList { get; }
```

### Bemerkungen

Wenn Werte von erweiterten Attributen als Eigenschaften von Elementen im Schema angegeben werden, können sie entweder durch Werte oder durch Verweise auf die in dieser Liste enthaltenen Werte angegeben werden. Anwendungen können die Reihenfolge der Liste durch die hier angegebene Reihenfolge übernehmen. Derzeit unterstützt für die Formate MSP 2003/2007 Xml und MSP 2003 mpp. Ändern Sie diese Liste nicht direkt. Verwenden Sie stattdessen die Methoden ExtendedAttributeDefinition.AddLookupValue/RemoveLookupValue.
