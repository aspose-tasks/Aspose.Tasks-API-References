## ExtendedAttributeDefinition.FieldId property

Erhält oder setzt die Projekt-ID eines benutzerdefinierten Felds. Verwenden Sie die Zeichenfolgendarstellung einer Konstante von[`ExtendedAttributeTask`](../../extendedattributetask/) Klasse zu spezifizieren`FieldId` Eigentum.

```csharp
public string FieldId { get; set; }
```

### Bemerkungen

Bevorzugte Art der Einstellung`FieldId` Eigentum ist zu schaffen[`ExtendedAttributeDefinition`](../) mit einer der dedizierten Factory-Methoden wie[`CreateTaskDefinition`](../createtaskdefinition/) oder[`CreateLookupTaskDefinition`](../createlookuptaskdefinition/) .

### Beispiele

```csharp
customFieldDefinition.FieldId = Aspose.Tasks.ExtendedAttributeTask.Number10.ToString("D");
```
