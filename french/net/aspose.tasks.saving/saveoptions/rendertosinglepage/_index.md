## SaveOptions.RenderToSinglePage property

Obtient ou définit une valeur indiquant si un projet doit être rendu sur une seule page lorsque le projet est enregistré au format graphique. La taille de la page sera modifiée afin que le projet rendu puisse tenir sur une seule page.

```csharp
public bool RenderToSinglePage { get; set; }
```
