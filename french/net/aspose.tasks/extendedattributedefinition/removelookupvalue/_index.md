## ExtendedAttributeDefinition.RemoveLookupValue method

Supprime une valeur de la liste de recherche interne. C'est un moyen préférable pour les manipulations avec le[`ValueList`](../valuelist/) .

```csharp
public void RemoveLookupValue(Value value)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| value | Value | Valeur à supprimer de la recherche. |

### Remarques

Cette méthode ne fonctionne que pour[`ExtendedAttributeDefinition`](../) instances qui ont[`CalculationType`](../calculationtype/) est égal àLookup .
