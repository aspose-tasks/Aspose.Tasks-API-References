## SaveOptions.RollUpGanttBars property

요약 작업 표시줄의 하위 작업을 표시해야 하는지 여부를 나타내는 값을 가져오거나 설정합니다. 하위 작업의 경우 롤업 필드는 하위 작업 Gantt 막대의 정보가 요약 작업 표시줄에 롤업되는지 여부를 나타냅니다. 요약 작업의 경우 롤업 필드는 요약 작업 표시줄에 롤업된 막대가 표시되는지 여부를 나타냅니다. 요약 작업에 대한 롤업 필드가 예로 설정되어 있어야 하위 작업이 롤업됩니다.

```csharp
public bool RollUpGanttBars { get; set; }
```
