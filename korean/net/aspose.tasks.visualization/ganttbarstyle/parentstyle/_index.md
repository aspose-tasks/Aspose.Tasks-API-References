## GanttBarStyle.ParentStyle property

사용자 지정 작업별 스타일에 대한 상위(또는 공통) 스타일을 가져오거나 설정합니다.

```csharp
public GanttBarStyle ParentStyle { get; set; }
```

### 비고

작업에는 상위 스타일이 다른 여러 사용자 지정 스타일이 있을 수 있습니다. 예를 들어 "중요" 상위 스타일이 있는 사용자 지정 스타일과 "보통" 상위 스타일이 있는 또 다른 스타일이 있는 작업을 고려하십시오. 간단히 말해 작업이 중요한 경우 첫 번째 스타일이 적용됩니다. 작업이 중요하지 않은 경우 두 번째 스타일이 적용됩니다(이 논리는 Microsoft Project Professional에서 상속됨).
